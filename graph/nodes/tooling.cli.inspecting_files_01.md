# View and search file contents from the command line

## Learning target

Print a file, page through a long one, and search within it for a pattern from
the terminal — the fast way to answer "what's in this file?" without opening an
editor. Capture a short transcript showing you viewing and searching a file, and
state what you found.

## Study pointers

The Missing Semester "The Shell" lecture (missing.csail.mit.edu/2020/course-shell)
for `cat`, `less`, and `grep`; on Windows, the Microsoft "Windows commands"
reference for `type`, `more`, and `findstr`
(learn.microsoft.com/windows-server/administration/windows-commands).

## Notes

Manual-gated: reading and interpreting what a file's contents mean is judgment,
not an exit code.
