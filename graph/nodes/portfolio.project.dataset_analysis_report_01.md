# Produce a dataset analysis report portfolio project

## Learning target

Pick a dataset, ask a real question of it, and answer that question with a
Pandas analysis — loading, grouping, and aggregating the data — then deliver a
report: the finding stated plainly, at least one chart that earns its place, an
honest account of what the data does and does not support, and the notebook or
script that reproduces the numbers.

## Study pointers

- Real Python and the Pandas docs for the group-and-aggregate analysis; the
  Fundamentals of Data Visualization book for making the chart honest and
  readable.
- Google Technical Writing guidance for the report prose — state each claim and
  the evidence under it.

## Notes

Hard-edged only from the code skills the analysis is built from (working with a
DataFrame and grouping/aggregating it) — mechanically enabling. The chart, the
prose, and the honest-reading discipline are the deliverable's packaging and
judgment, so visualization, communication, and statistics edges are soft.
Manual-gated: a working analysis plus a readable report plus an honest write-up
is judged, never proxied.
