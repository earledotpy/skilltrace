# Solve a mixed math-foundations problem set

## Learning target

Work a set of problems that deliberately mixes percentages and ratios, order
of operations, solving linear equations, and reading linear functions — without
the section headings that tell you which technique to reach for. The skill this
weaves is *selection*: recognising from a bare problem which foundation applies,
then applying it cleanly. Show the work, not just the answers, so the reasoning
is visible.

## Study pointers

- Khan Academy Arithmetic and Algebra — use the mixed-review and unit-test
  exercises rather than a single topic drill; those are already interleaved.
- Interleaving is the point: pull a few problems each from percentages/ratios,
  linear equations, and linear functions and shuffle them before solving.

## Notes

Soft-edged from the bands it weaves — a learner who already has these
foundations can pass it directly. Manual-gated: integration quality (did you
pick the right tool and reason cleanly?) is judgment, not a single correct
output.
